# 🗄️ Project 3 — SQL Data Analysis

## 📌 Objective
Use SQL queries to extract insights from the 
e-commerce dataset using SELECT, WHERE, 
GROUP BY and ORDER BY commands.

---

## 📁 Dataset Info
- **Source:** Cleaned e-commerce dataset
- **Rows:** 1200
- **Columns:** 14

---

## 🔍 SQL Queries Performed

### 1. SELECT Query
- Fetched top 5 rows from dataset

### 2. COUNT — Orders by Product
- Counted total orders per product
- Printer had highest orders (181)

### 3. SUM & AVG — Revenue by Product
- Calculated total & average revenue
- Printer generated highest revenue

### 4. WHERE Filter
- Filtered only Delivered orders
- Sorted by highest TotalPrice

### 5. GROUP BY Payment Method
- Counted orders by payment type
- Calculated total revenue per method

---

## 📊 Key Findings
- Printer — Highest revenue product
- Most orders were Cancelled (250)
- Average Order Value: ₹1053.97
